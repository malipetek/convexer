# Convexer Promo Site

This is the promotional landing page for Convexer, deployed to GitHub Pages.

## Development

```bash
cd promo
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```

## Deployment

The site is automatically deployed to GitHub Pages via the `.github/workflows/promo.yml` workflow when changes are pushed to the `main` branch in the `promo/` directory.

The site will be available at: `https://malipetek.github.io/convexer/`
